# analisis-campa-as
# Análisis de Keywords y Rendimiento de Campañas

Este proyecto realiza un proceso de ETL (Extracción, Transformación y Carga) sobre un dataset de métricas de marketing digital.

## Funcionalidades
- **Limpieza de datos:** Identificación y eliminación de duplicados en keywords.
- **Análisis estadístico:** Visualización de distribuciones con histogramas y comparativas con gráficos de barras (Seaborn).
- **Filtrado SQL:** Uso de `pandasql` para segmentar campañas con alto impacto (Impresiones >= 3500).
- **Exportación:** Generación de un dataset limpio para reporting.

## Tecnologías
- Python (Pandas, Seaborn, Matplotlib)
- SQL (via pandasql)
